What Is The Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents, primarily HTML and XML. It represents the structure of a document as a tree of objects, where each element in the document is treated as a node, allowing programming languages like JavaScript to interact with the document's structure, style, and content.

What is event delegation in the context of the DOM,and why is it useful?

Event delegation is a technique in the DOM that allows you to handle events more efficiently by taking advantage of event bubbling. Instead of attaching event listeners to individual child elements, you attach a single event listener to a parent element. This listener can handle events triggered by any of its child elements through event propagation

To manipulate an element's attributes and styles using the DOM, you can use JavaScript methods and properties that directly interact with the element.
